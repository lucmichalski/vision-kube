Usage
-----

First install the server dependencies:

    $ cd server
    $ npm install

Next install all frontend dependencies using [Bower](http://bower.io):

    $ cd client
    $ bower install

Run the server:

    $ coffee server.coffee

Navigate to the homepage in your browser to try out the webservice:

    http://localhost:3000/
